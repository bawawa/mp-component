# 小程序组件

### 1.刮刮乐组件

scratch

| 参数         | 类型     | 默认值  | 说明            |
|------------|--------|------|---------------|
| width      | string | 300  | 宽度（单位px）      |
| height     | string | 150  | 高度（单位px）      |
| coverColor | string | #999 | 刮刮乐覆盖颜色       |
| prizeName  | string | 一等奖  | 刮刮乐文字（支持\n换行） |
